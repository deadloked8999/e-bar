# E-Bar Backend API

Backend для системы управления документами цифрового бара/клуба/гостиницы.

## Установка

1. Создайте виртуальное окружение:
```bash
python -m venv venv
```

2. Активируйте виртуальное окружение:
```bash
# Windows
venv\Scripts\activate

# Linux/Mac
source venv/bin/activate
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Создайте файл .env (скопируйте .env.example)

## Запуск

```bash
python main.py
```

API будет доступно по адресу: http://localhost:8000

Документация API: http://localhost:8000/docs

## Структура документов

### Блок 1 - Регистрационные документы
- ОГРН/ИНН
- Устав
- Свидетельство о регистрации

### Блок 2 - Лицензии
- Лицензия на продажу алкоголя
- Лицензия на общепит
- Санитарное разрешение

### Блок 3 - Финансовые документы
- Выписка из банка
- Налоговая декларация
- Договор аренды помещения

### Блок 4 - Документы представителя
- Паспорт директора
- Приказ о назначении
- Доверенность

## API Endpoints

- `POST /api/documents/upload` - Загрузка документа
- `GET /api/documents` - Получить все документы
- `GET /api/documents/{id}` - Получить документ по ID
- `POST /api/documents/{id}/verify` - Верифицировать документ
- `DELETE /api/documents/{id}` - Удалить документ
- `GET /api/documents/stats` - Статистика по документам
